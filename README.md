# supergulp:dolphin:
Frontend portfolio for web development. Using GULP. 
Gulp는 Node.js 기반의 프로세스 자동화 도구이며 MIT 라이센스의 오픈소스 입니다. 
이 프로젝트는 간편한 퍼블리싱 작업을 위해 개발되었습니다.

## Installation
1. node.js 설치: [node.js]https://nodejs.org/en/download/
2. npm(node module) 설치
``` js
npm install
```
